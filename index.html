1. Что такое margin and padding? Margin и padding - это свойства CSS, которые используются для управления пространством вокруг элемента. Margin - это пространство между элементом и его внешними границами. Padding - это пространство между элементом и его содержимым.
2. Что такое block, inline, inline-block? Block, inline и inline-block - это типы блочных элементов в CSS. Block элементы располагаются в строку, но занимают всю ширину доступного пространства. Inline элементы располагаются в строке и не занимают всю ширину доступного пространства. Inline-block элементы сочетают в себе свойства block и inline элементов. Они располагаются в строке, но могут занимать всю ширину доступного пространства.
3. CSS overflow CSS overflow - это свойство, которое определяет, что происходит с содержимым элемента, которое выходит за его границы. visible - содержимое элемента полностью отображается. hidden - содержимое элемента полностью скрыто. scroll - содержимое элемента отображается с помощью полосы прокрутки. auto - содержимое элемента отображается с помощью полосы прокрутки только в том случае, если оно выходит за границы элемента.
4. CSS float CSS float - это свойство, которое позволяет элементам располагаться рядом друг с другом, независимо от их естественного потока. left - элемент будет выровнен по левому краю. right - элемент будет выровнен по правому краю.
5. CSS @media rule CSS @media rule - это правило, которое позволяет определять стили для определенных устройств или разрешений экрана.
